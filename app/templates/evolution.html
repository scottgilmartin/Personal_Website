ABOUT: Evolution project.
(Use the arrows to cycle through the images.)
A Python code was written as a simple evolution/ artificial selection simulator. Above are a selection of randomly generated sprites using the simulator, where the last image shows the barcode like 'DNA' from which they are generated. we generate sprites based on mutations of an original 'DNA' matrix. Every sprite has an associated barcode like DNA from which it is drawn by applying the rules of a modified version of Conway's game of life. By performing small mutations on this DNA we can draw a set of similar but distinct sprites. For more info, source code and to see the evolution process see my GitHub: GitHub
