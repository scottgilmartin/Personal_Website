ABOUT: Mood rings project.
(Use the arrows to cycle through the chapters.)
A Python code was written to analyse a script for interactions between characters, to extract the emotional content of each interaction, and to represent this in the form of a graph (using networkx/ nxviz). In the images above, each line (edge) indicates an interaction between two characters (nodes). In this case we say that when two characters have multiple consecutive lines (with other checks to minimize false positives) they are interacting. Emotional content of a given line was estimated using data from: http://hedonometer.org/words.html. Above, blue indicates positive interactions and red indicates negative interactions. The result allows us to see tension and resolution as different characters interact throughout the story.
