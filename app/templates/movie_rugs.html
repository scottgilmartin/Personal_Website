ABOUT: Movie rugs project.
(Use the arrows to cycle through the movies.)
A Python code was written to analyse a script and print a square of a particular colour every time a character name appears in the script using Pillow. Each character is assigned a colour of the user's choice. The resulting image is a "barcode" for the movie, where we can see which characters are interacting at any given point in the movie.
