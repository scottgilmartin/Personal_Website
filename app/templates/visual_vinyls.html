ABOUT: Visual vinyls project.
(Use the arrows to cycle through the vinyls.)
A Python code was written to generate a circular graph where nodes represent beats in a bar and edges represent a movement from a melody note played on its tail node to a note played on its head node. The edges follow the melody of the lead vocal part of the track. The input is a list of the melody note occurrences; for example if the first melody note is on the second beat of the bar, and the second melody note is on the third beat of the bar, the input list would begin with [2,3,...]. The result shows the rhythmic signature of the first few bars of the song.
